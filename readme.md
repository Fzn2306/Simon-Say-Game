#🎮 Simon Says Game

A browser-based implementation of the classic Simon Says memory game built using HTML, CSS, and JavaScript. Test your memory by repeating an increasingly longer sequence of colored buttons. The game becomes more challenging with each level and keeps track of your highest score.

📌 Features
🎲 Random color sequence generation
🧠 Memory-based gameplay
📈 Progressive difficulty with each level
✨ Button flash animations for game and user input
✅ Real-time sequence validation
🏆 High score tracking
❌ Game over screen with visual feedback
📱 Clean and responsive user interface


🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)


🚀 How to Play
Open the game in your browser.
Press any key to start.
Watch the button that flashes.
Repeat the sequence by clicking the colored buttons in the correct order.
Each successful level adds a new color to the sequence.
Continue until you make a mistake.
Try to beat your highest score!
